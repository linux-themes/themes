## Install

```sh
	go install github.com/linux-themes/themes@latest
```

## Use

```sh
	themes list all
	themes install 21 https://github.com/sudo-adduser-jordan/mint-y-winx/raw/main/mint-y-winx.tar.xz
	themes remove 
```
